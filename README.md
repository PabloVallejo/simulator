# README

### Getting started

```bash
$ git clone https://github.com/PabloVallejo/simulator.git

$ cd simulator 

# Build the project .
$ docker-compose build 

# Create the databases.
$ docker-compose run web rails db:create

# Run the project.
$ docker-compose up
```

Go to [http://localhost:3000](http://localhost:3000) to see the project
